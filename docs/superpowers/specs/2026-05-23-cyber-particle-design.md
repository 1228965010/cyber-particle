# Cyber Particle — 赛博朋克手势粒子系统

## 概述

纯网页应用，通过摄像头实时识别手部手势，控制 15000+ WebGL 粒子在赛博朋克视觉风格下的形态变换与位置追踪。

## 技术栈

| 层 | 方案 |
|----|------|
| 手势检测 | MediaPipe Hands (CDN, 21关键点) |
| 手势分类 | 自定义几何算法（手指弯曲度 + 方向） |
| 渲染 | Three.js + PointsMaterial / BufferGeometry |
| 后处理 | CSS CRT扫描线 + 自定义 Shader 辉光 |
| 部署 | Vercel 或 GitHub Pages |

## 架构

```
摄像头帧 → MediaPipe Hands → landmarks[21]
  ├─ 手指角度计算 → 手势分类（握拳/张开/捏合/食指指向）
  └─ 手掌中心坐标 → 粒子引力/斥力场中心
         ↓
  Three.js 粒子系统（每帧更新）
```

## 位置追踪

手掌中心作为引力场源，15000 粒子实时受力偏移，带阻尼惯性衰减。

## 视觉设计

- 暗黑深渊背景 + 淡六边形网格线
- HUD: 左下角手势名（等宽字体、青色、打字机光标）、底部 4 段状态条
- 角落系统状态（FPS、粒子数、置信度）
- CRT 扫描线 (CSS repeating-linear-gradient)
- 屏幕暗角 (radial-gradient vignette)
- 粒子 additive blending 辉光

## 边界情况

- **无摄像头权限**: 静态粒子演示 + 提示
- **摄像头占用**: 检测 NotReadableError，提示用户
- **画面无手**: 粒子进入待机漂浮模式
- **低光照**: 置信度低时粒子行为降级为惯性运动
- **低端设备**: 手动/自动降档 (15000 → 5000 → 2000)
- **后台标签页**: rAF 自动暂停
- **手势抖动**: 200ms 防抖 + 置信度阈值 0.7

## 手势识别逻辑

基于 MediaPipe Hands 21 个关键点，计算每个手指的弯曲角度：

- **握拳**: 五指指尖到指根距离 < 阈值
- **张开**: 五指指尖到指根距离 > 阈值，且指尖分散
- **捏合**: 拇指尖与食指尖距离 < 阈值，其余手指张开
- **食指指向**: 食指伸直，其余手指弯曲

## 不纳入范围

- 人脸检测
- 多手支持（只追踪第一只手）
- 音频响应
- 移动端适配（优先桌面端）
